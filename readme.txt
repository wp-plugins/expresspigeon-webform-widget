ExpressPigeon webform widget displays ExpressPigeon webform using iframe.

== Description ==
ExpressPigeon webform widget displays ExpressPigeon webform using iframe.

== Installation ==
1. Download and unzip ep-webform-widget.zip
2. Upload the folder containing `ep-webform-widget.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. To add an ExpressPigeon webform on a sidebar, browse to `Appearance > Widgets` and add the `ExpressPigeon webform widget` to desired sidebar. Configure webform GUID, height, width and save your changes.

== Frequently Asked Questions ==

= Can I add explicit style to created iframe? =

You can specify explicit styles for iframe in CSS style input in widget configuration

= How do I add an ExpressPigeon webform to a blog-post or Page? =

Login into your ExpressPigeon account, grab 'Iframe content' from your web form and insert it into your post.
